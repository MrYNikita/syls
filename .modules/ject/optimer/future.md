- [ ] Ускоритель
    - [ ] Ускорение поиска позиции по строчным свойствам элементов массива за счёт алфивитных групп. Адаптивность с картами алфавитных групп.
    - [ ] Ускорение поиска позиции по числовым свойствам элементов массива за счёт меток средних значений. Адаптивность с картами меток средних значений.
- [ ] Минимизатор
    - [ ] Возможность конвертировать массив чисел в функцию, возвращающую в качестве результата массив с теми эе значениями в неупорядоченном виде.
    - [ ] Возможность конвертировать числовые массивы в диапазоны значений.
    - [ ] Возможность превращения слов разных размеров (в том числе и не осмысленных) в числа.
    - [ ] Возможность превращения объектов в числа.
